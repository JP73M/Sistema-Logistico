Objetivo

Desarrollar un sistema de control operativo logístico para la gestión y clasificación de mercancías mediante lectura de guías, integración con báscula, control de manifiestos, gestión de casilleros, historial operativo y trazabilidad de usuarios.

Versión 1.0

- Login
- Carga de manifiestos
- Carga de casilleros
- Escaneo de guías
- Gestión de peso
- Historial
- Exportación Excel

Visión futura

- Multiusuario
- Multi sucursal
- Aplicación web centralizada
- Dashboard avanzado
- Reportes operativos
